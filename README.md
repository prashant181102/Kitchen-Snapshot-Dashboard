# Kitchen-Snapshot-Dashboard
A Power BI dashboard project designed to analyze Kitchen Revenue, EBITDA, Gross Margin, Variance, and Store Performance across multiple cities and stores. This dashboard provides operational and financial insights through interactive visualizations for business monitoring and decision-making.
# Project Overview : 
This dashboard helps track and evaluate kitchen business performance using KPIs and visual analytics.
The report contains two major sections:
Kitchen Snapshot Dashboard
Revenue & profitability overview
Store-wise and city-wise analysis
Revenue trends and EBITDA performance
Variance by Kitchen Dashboard
Variance monitoring across stores and cities
Revenue category comparisons
Monthly and yearly variance analysis
